O que o Script está fazendo?

Está editando o simples HTML abaixo (que é o arquivo do atual).

   <h1>DOM</h1>

   <p><strong>D</strong>ocument <strong>O</strong>bject <strong>M</strong>odel</p>

   <p>Aqui vai o resultado</p>

   <p>Aprendendo a usar o <strong>DOM</strong> em JavaScript.</p>

   <div id="msg">Texto Original</div>

O arquivo possuí uma folha de estilos aplicando a cor amarela para todos os elementos <p>.

    p{
        color: yellow;
    }

Primeiramente, declarando 3 variáveis (p1, p2 e p3) e armazenando nelas, o conteúdo dos elementos capturados pelo nome da TAG.

        var p1 = window.document.getElementsByTagName('p')[0];
        var p2 = window.document.getElementsByTagName('p')[1];
        var p3 = window.document.getElementsByTagName('p')[2];

O conteúdo de cada variável, agora recebe um estilo de cor diferente.

        p1.style.color = 'blue';
        p2.style.color = 'green';
        p3.style.color = 'black';

O conteúdo da TAG <div>, que possui um id="msg" agora é armazenado na variável (d1) usando a função getElementById. Também é aplicado um estilo de cor diferente no background "vermelho" e seu texto é alterado para 'Mudando o conteúdo', usando a função innerText.

        var d1 = window.document.getElementById('msg');
        d1.style.background= 'red';
        d1.innerText = 'Mudando o conteúdo';

Mas agora o conteúdo busca o primeiro elemento <div> que possui o atributo id com o valor 'msg'. Altera a cor do texto para azul e a cor de fundo para preto.

        var d1 = window.document.querySelector('div#msg');
        d1.style.color = 'blue';
        d1.style.background = 'black';
